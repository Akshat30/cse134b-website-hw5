some of my most notable projects as a computer science student and as a software engineer.

project 1: homespace housing platform

This project was developed with a few friends, and we had seeked to solve the housing crisis that current college students at UCSD face every year. It is a website that allows landlords in the San Diego area to post listings for college students. It also has an ID verification system to guarantee that only college students are able to lease from this website.

technologies used: Prisma, tRPC, MongoDB, AWS for backend, and Next.js (React) for the front-end.

project 3: verum large language model text analysis

This project allows a given user to choose a famous political speech, or input their own, and receive a detailed analysis of it. This includes a general analysis, specific fact checking, an unbiased summary, and more. It extracts the text from a given article, uses the Google API to receive additional information, summarizes the text, and uses OpenAI's LLM endpoints to receive an analysis.

technologies used: NodeJS, Next.js, Google API, OpenAI API, AWS
